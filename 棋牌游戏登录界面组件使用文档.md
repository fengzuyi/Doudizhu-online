# 棋牌游戏登录界面组件使用文档

本文档说明如何在 React 项目中使用「舒缓风棋牌游戏登录界面」组件。

设计稿来源：Figma 登录页设计  
适用场景：斗地主、麻将、好友房、棋牌游戏大厅登录前页面

---

## 一、推荐项目结构

建议在项目中创建以下文件：

```bash
src/pages/LoginPage.jsx
src/pages/LoginPage.css
```

如果你的项目已经有 `pages` 目录，可以直接放进去。  
如果没有，也可以放在：

```bash
src/components/LoginPage.jsx
src/components/LoginPage.css
```

## 五、配合 React Router 使用

如果你的项目使用了 `react-router-dom`，可以这样配置：

```jsx
import { BrowserRouter, Routes, Route } from "react-router-dom";
import LoginPage from "./pages/LoginPage";
import GameHall from "./pages/GameHall";

export default function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<LoginPage />} />
        <Route path="/hall" element={<GameHall />} />
      </Routes>
    </BrowserRouter>
  );
}
```

## 七、接入真实登录接口

当你有后端登录接口后，可以这样写：

```jsx
const handleLogin = async () => {
  const res = await fetch("http://localhost:3000/api/login", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      username,
      password,
    }),
  });

  const data = await res.json();

  if (data.success) {
    localStorage.setItem("token", data.token);
    navigate("/hall");
  } else {
    alert(data.message || "登录失败");
  }
};
```

## 八、把输入框改成受控组件

如果要接入登录接口，需要先引入 `useState`：

```jsx
import { useState } from "react";
```

然后在组件中添加状态：

```jsx
const [username, setUsername] = useState("");
const [password, setPassword] = useState("");
```

把输入框改成：

```jsx
<input
  type="text"
  placeholder="请输入手机号 / 游戏账号"
  value={username}
  onChange={(e) => setUsername(e.target.value)}
/>

<input
  type="password"
  placeholder="请输入密码"
  value={password}
  onChange={(e) => setPassword(e.target.value)}
/>
```

## 九、登录组件和 Figma 的关系

Figma 负责设计界面长什么样。

React 负责让这个界面真正运行起来。

所以项目中的实际使用方式是：

1. 参考 Figma 设计稿
2. 在 React 中创建 `LoginPage.jsx`
3. 在 CSS 中还原视觉样式
4. 绑定登录按钮、游客试玩、微信登录等事件
5. 接入后端接口
6. 登录成功后跳转到游戏大厅
